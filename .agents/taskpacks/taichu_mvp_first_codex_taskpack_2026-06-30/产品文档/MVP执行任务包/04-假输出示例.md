# 04-假输出示例

> 更新日期：2026-06-30

假输出只用于检查输出结构、链路和提示词快照，不代表真实模型能力。

## 1. 通用结构

```json
{
  "message_id": "msg_0001",
  "role": "assistant",
  "task_type": "continue",
  "subtask_type": null,
  "reference_scope": "chapter",
  "output_type": "text_candidate",
  "is_mock": true,
  "content": {},
  "source_refs": [],
  "prompt_snapshot": {
    "structured": {},
    "final_prompt": "【模拟提示词】"
  },
  "created_at": "2026-06-30T12:00:00+09:00"
}
```

前端显示“模拟输出”，不要直接显示 `is_mock`。

## 2. 续写

只返回正文候选文本，不返回解释。

```json
{
  "message_id": "msg_continue_001",
  "role": "assistant",
  "task_type": "continue",
  "reference_scope": "chapter",
  "output_type": "text_candidate",
  "is_mock": true,
  "content": {
    "text": "秦阳立在残碑之前，指尖的金色鳞光一点点沉入掌心。山风从破碎的石阶间穿过，像有无数人在低声念诵一个早已失传的名字。"
  },
  "source_refs": [],
  "prompt_snapshot": {
    "structured": {
      "user_input": "续写 200 字，压迫感更强",
      "reference_scope": "chapter",
      "chapter_title": "第1章 大田金鳞元神出",
      "route": "mock_continue_text"
    },
    "final_prompt": "【模拟提示词】任务：续写。参考范围：本章。作者要求：续写 200 字，压迫感更强。"
  }
}
```

## 3. 润色

子类型为扩写、缩写、改写。只保存结果。

```json
{
  "message_id": "msg_polish_001",
  "role": "assistant",
  "task_type": "polish",
  "subtask_type": "rewrite",
  "reference_scope": "selection",
  "output_type": "text_candidate",
  "is_mock": true,
  "content": {
    "text": "他没有立刻开口，只是抬眼望向那座沉在云后的山门。"
  },
  "source_refs": [],
  "prompt_snapshot": {
    "structured": {
      "user_input": "改得更有天命感",
      "selected_text": "他看向山门。",
      "route": "mock_polish_rewrite"
    },
    "final_prompt": "【模拟提示词】任务：润色-改写。参考范围：选区。"
  }
}
```

## 4. 设定

```json
{
  "message_id": "msg_setting_001",
  "role": "assistant",
  "task_type": "setting",
  "reference_scope": "chapter",
  "output_type": "setting_result",
  "is_mock": true,
  "content": {
    "setting_addition": "可以把金鳞异象设定为元神外显的一种早期征兆，但暂不确认它属于固定境界规则。",
    "usage_suggestion": "正文中先用感官和旁人反应暗示，不要直接解释完整体系。",
    "possible_impact": "若后续确认为境界规则，需要补充到境界卡或规则卡。"
  },
  "source_refs": [],
  "prompt_snapshot": {
    "structured": {
      "user_input": "补一下这个异象的设定方向",
      "route": "mock_setting_result"
    },
    "final_prompt": "【模拟提示词】任务：设定。参考范围：本章。"
  }
}
```

右侧“设定”入口第一版不允许一键转成待确认事实。

    "conclusion": "当前没有可确认的正式依据，只能作为推测。",

## 7. 章节摘要

章节摘要不进入普通 AI 多轮对话记录。

```json
{
  "chapter_id": "chapter-001",
  "is_mock": true,
  "content": {
    "chapter_overview": "本章主要展示主角遭遇金鳞异象，并引出元神相关悬念。",
    "character_changes": "主角第一次意识到自身异象可能与普通修行不同。",
    "plot_progress": "异象出现后，外界势力开始注意主角。"
  },
  "prompt_snapshot": {
    "structured": {
      "task_type": "chapter_summary",
      "reference_scope": "chapter",
      "route": "mock_chapter_summary"
    },
    "final_prompt": "【模拟提示词】任务：章节摘要。参考范围：本章。"
  }
}
```
