# 10-前端文案与导航清单

> 更新日期：2026-06-30

## 1. 主导航

写作、知识库、Inbox、AI 历史、智能体工作台、设置。

智能体工作台显示：实验功能。

## 2. 写作区入口

清晰名：纯对话、续写、润色、设定、建议、证据、章节摘要、灵感、事实。

品牌化名：问灵、衍文、润笔、构界、策议、溯源、章要、灵引、事实簿。

品牌化模式下，“AI”显示为“器灵”。命名体系切换不持久化，刷新后回到清晰名。

## 3. AI 面板文案

通用：正文参考、当前参考、发送、查找对话、新对话、模拟输出、查看提示词快照、收起提示词快照。

占位文案：

| 入口 | 文案 |
|---|---|
| 纯对话 | 输入你想临时询问的内容 |
| 续写 | 输入续写要求，可为空 |
| 润色 | 输入扩写、缩写或改写要求，可为空 |
| 设定 | 输入你想补充的设定方向 |
| 建议 | 输入你想判断或改进的问题 |
| 证据 | 输入你想追问的依据或出处 |
| 灵感 | 记下一条灵感 |
| 事实 | 记下一条可能入库的事实 |

章节摘要：本章暂未生成摘要、当前章节为空，无法生成摘要、正在生成章节摘要……、器灵正在整理章要……、章节摘要生成失败。

## 4. 编辑器工具栏

允许：字体大小、字体样式、编辑背景、撤销、恢复、查找、一键段落排版、保存。

禁止：正文、标题一、标题二、引用、替换。

保存状态：未保存、保存中、已保存、保存失败、已保存 HH:mm、自动保存失败，请手动保存。

## 5. Inbox

Tab：灵感、待确认事实、待处理问题。

状态：待处理、已处理、已废弃。

动作：编辑、标记已处理、废弃、确认入库、查看知识卡、留在 Inbox。

详情：原事实、知识卡预览。

## 6. 知识库

类型：角色、境界、功法、地点、势力、物品、规则、事件、伏笔。

筛选：全部、草稿、有效、只看废弃。

按钮：新建知识卡、新建当前类型、保存、标记为有效、标记废弃、添加来源。

空状态：当前类型暂无知识卡；新建当前类型知识卡。

字段中文名：名称、别名、摘要、正文补充、重要程度、状态、来源引用、身份、所属势力、当前境界、功法、物品、关系摘要、出场章节、状态记录。

禁止裸露：`id`、`type`、`aliases`、`source_refs`、`created_at`、`updated_at`、`task_type`、`output_type`。

## 7. AI 历史

列表字段：时间、章节、功能入口、模型、是否错误、是否有来源、首条用户输入摘要、模拟输出。

筛选：章节、功能入口、是否有来源、是否有错误。

详情：消息内容、本轮任务类型、参考范围、输出类型、来源引用、提示词快照。

## 8. 设置页

允许：编辑器偏好、字体大小、字体样式、编辑背景、基础显示偏好。

禁止：模型配置、API Key、服务地址、供应商、温度、token、真实 LLM。
