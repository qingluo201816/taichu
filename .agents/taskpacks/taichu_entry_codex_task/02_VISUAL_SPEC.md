# 视觉规格

## 核心气质

玄幻、浩瀚、暗而不空、满而不炸、克制、高级、超现实。不是赛博朋克，不是游戏 HUD，不是星空壁纸。

## 正确画面

第一人称站在一个点云化的玄幻位面入口前。下半屏是点云地面，中部是横向粒子雾带，左右是弱包裹的树墙/山影/位面边界，上方是有深度的暗色点云天空，远景宫阙从地平线附近若隐若现。

## 错误画面

1. 星空里漂一个门。
2. 中央一个粒子图标。
3. 前景底部糊成白色棉花团。
4. 上半部大面积纯黑留白。
5. 天门点击后才出现。
6. 粒子被线条连接成神经网络。
7. 纯随机撒点，没有地面/边界/远景结构。

## 构图比例

- 地平线：画面高度 58%–63%。
- 地面层：下半屏 50%–60%。
- 宫阙：宽 18%–22%，高 8%–10%，位置略高于地平线。
- TAICHU：顶部中央，宽约 180–260px，高约 48–70px，细边框。

## 色彩

- 背景：极深暖黑/炭黑。
- 主粒子：暖白、灰白、象牙白。
- 辅助粒子：少量暗墨绿、灰绿、淡金。
- 焦点微粒：极克制暗绿 glow。
- 禁止：大面积纯白、霓虹蓝紫、强赛博绿、全局泛黄。
