# 参考站可借鉴点

## Penderecki's Garden 可借鉴

- 它把真实园林转译为虚拟点云空间，而不是普通粒子背景。
- 官方项目页描述其使用先进的空间激光扫描，把音乐、自然声音、树木等内容置入数字园林空间。
- Awwwards 技术说明中提到：前端为 TypeScript + Three.js，3D 数据以 PCD 点云格式存储并 gzip 压缩，使用 GLSL shader 让粒子自然运动，Web Audio API 影响点的微动，转场非常重视平滑性。
- 它的章节/空间结构很强：用户不是看一个图，而是在点云场景里探索章节。

## 太初应参考什么

- 参考“点云资产/结构驱动”的实现思路。
- 参考“进入一个虚拟空间”的仪式感。
- 参考“深色、低饱和、自然点云、平滑镜头推进”。
- 参考“可访问性”：键盘可用、低动效模式、对比度/文本兜底。

## 太初不要参考什么

- 不复制 Penderecki 的园林、房子、文案、音乐内容。
- 不做真实园林，不做波兰花园风格。
- 不做右上角坐标、菜单和章节结构的照搬。
- 不把宫阙做成普通房子。

## 太初自己的核心隐喻

最高位面宫阙只是“万界之上的远景锚点”。用户点击 TAICHU 后，不是进入宫阙，而是穿过最高位面的粒子层，最终被吸入一个不起眼的小粒子，象征从无数大千世界中进入一部具体小说的创作现场。
