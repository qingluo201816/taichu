# 产品上下文

> 更新日期：2026-06-27

太初是面向单个作者、单本玄幻长篇小说的个人 AI 创作 IDE。它不是通用写作软件、不是发布平台、不是协作 SaaS、不是一键自动写书工具。

MVP0.1 的核心闭环是：

进入当前章节 → 正常写作 → 选中文本触发 AI 问答/设定补充/续写 → AI 以卡片形式输出 → 作者插入正文、保存灵感、生成待确认设定或丢弃 → 作者确认后进入知识库 → 下一次检索能引用来源。

本任务发生在 MVP0.1 功能闭环完成后，目标是统一前端视觉体验。它不改变业务功能，不改变数据事实源，不改变 API 语义。

## 视觉目标

前端必须体现三层设计系统：

1. 入口虚空层：万象未分、诸界未名，负责进入太初的仪式感。
2. 深色写作工作台层：长时间写作环境，承载导航、编辑器外框、AI 面板、知识库、收件箱。
3. 纸质稿件内容层：正文、稿件、候选正文、阅读区域，强调中文长文可读性。

## 产品优先级

视觉改造必须服务：

- 正文推进
- 设定一致性
- 灵感沉淀
- 上下文维护
- 防 AI 污染
- 数据安全

任何视觉修改如果让作者更难找到正文、AI 卡片、待确认设定、来源证据，都应回退。
