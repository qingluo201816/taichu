# MVP0.1 完成后的假设

> 更新日期：2026-06-27

本任务假设 MVP0.1 功能已完成或基本完成，至少存在：

- 主工作台或首页
- 编辑器页面
- 章节导航
- AI 卡片或 AI 面板
- 收件箱或待处理页面
- 知识库页面
- 基础对话或查询页面

如果执行时发现这些页面不存在，不要创造完整业务功能，只能创建样式容器或占位，并明确标注“等待 MVP0.1 功能接入”。

本任务不负责实现：

- 章节保存逻辑
- AI 调用
- AI 卡片状态机
- PendingFact 确认逻辑
- 知识库写入
- 检索
- 导出
- 后端 API

若样式改造需要这些业务能力才能继续，必须停止并向用户说明阻塞点。
