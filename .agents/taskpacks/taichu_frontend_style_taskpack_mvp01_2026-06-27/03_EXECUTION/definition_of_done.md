# Definition of Done

> 更新日期：2026-06-27

本任务整体完成需要满足：

1. `STYLE_LOCK.md` 已由用户确认。
2. 三层设计系统已落到 token 和主要页面。
3. 深色工作台层覆盖主应用外壳。
4. 纸质稿件层覆盖编辑器正文和确认范围内的长文本区域。
5. AI 卡片、收件箱、知识库、证据展示风格统一。
6. 响应式和 reduced motion 基本可用。
7. `docs/太初前端视觉冻结说明.md` 已生成或更新。
8. `npm run lint` 和 `npm run build` 已运行并记录结果。
9. 若 `npm run test:editor` 可用，已运行并记录结果。
10. 未修改后端、project_assets、业务契约。
11. 用户可以按照验收清单手动检查。
12. 后续视觉变更只能是 bug、性能、适配和可访问性修复，不能随意换风格。
