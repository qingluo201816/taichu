# 停止条件

> 更新日期：2026-06-27

遇到以下情况必须停止并询问用户：

1. 找不到 `TAICHU_DESIGN.md`。
2. `TAICHU_DESIGN.md` 内容与用户描述明显不一致。
3. `docs/rule.md` 不存在或规则冲突。
4. Phase 0 未获得用户批准。
5. 当前 MVP0.1 页面不存在，且无法判断本次改造范围。
6. 需要修改后端才能完成样式任务。
7. 需要修改 `project_assets`。
8. 需要改变业务数据结构、API、状态机。
9. 需要新增重型 UI/动画库。
10. 当前 web build 已经失败且原因与本任务无关。
11. 样式改造会破坏编辑器输入、保存、AI 卡片操作。
12. 设计 token 无法从 `TAICHU_DESIGN.md` 提取，需要用户补充。
13. 用户要求与 Anti-goals 冲突。

遇到以下情况可以继续，但必须记录：

- 存在既有 lint 警告。
- 某些页面仍是占位页。
- 某些组件尚未接真实数据。
- 无法截图，但可提供手动验收路径。
