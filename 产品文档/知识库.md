# 知识库（D 类功能）

> 更新日期：2026-06-25
>
> 所属层级：**核心层（MVP 必做）**
>
> 关联文档：[架构设计](./架构设计.md)、[内容生成](./内容生成.md)

## 概述

知识库是独立管理页面，涵盖玄幻小说写作的六个核心实体维度。AI 写作时自动检索相关知识库内容并引用。

---

## D1 角色卡

| 字段 | 类型 | 说明 |
|------|------|------|
| 姓名 | 文本 | 必填 |
| 别名/称号 | 文本 | |
| 种族 | 文本 | |
| 性别 | 选择 | |
| 修为时间线 | 列表 | `[{"时间段":"第1-100章", "等级":"炼气期"}, ...]` |
| 势力归属时间线 | 列表 | `[{"时间段":"开局", "势力":"青云宗"}, ...]` |
| 性格标签 | 标签列表 | |
| 外貌描述 | 长文本 | AI 写作时引用 |
| 背景故事 | 长文本 | |
| 备注 | 长文本 | |
| 头像/立绘 | 图片 | **远期**，留给 DX-1 虚拟形象 |

## D2 世界观

分类笔记模式，每个分类下多条笔记（标题 + 自由文本）：

| 分类 | 示例 |
|------|------|
| 世界基本规则 | 灵力体系、天道规则 |
| 地理 | 大陆分布、禁地 |
| 历史 | 上古大战、纪元更迭 |
| 势力分布 | 宗门/国家/种族地盘 |
| 文化/种族 | 风俗、语言、禁忌 |

## D3 功法百科

| 字段 | 类型 | 说明 |
|------|------|------|
| 名称 | 文本 | 九转玄功 |
| 类型 | 选择 | 功法/战技/法术/秘术 |
| 品阶 | 文本 | 天阶/地阶/玄阶/黄阶 |
| 等级描述 | 文本 | 功法自身的分层（第一层→第九层） |
| 效果 | 长文本 | 修炼后的能力 |
| 所属势力/来源 | 文本 | |
| 备注 | 文本 | |

**注意**：功法百科记录世界中的功法。角色对功法的掌握情况记录在角色卡中。

## D5 时间线

按时间顺序的事件列表，关联角色/势力/地点：

| 字段 | 类型 | 说明 |
|------|------|------|
| 时间点 | 文本 | 如"第一卷/第5章" |
| 事件 | 文本 | 事件描述 |
| 关联角色 | 多选 | |
| 关联地点 | 多选 | |
| 关联势力 | 多选 | |

## D6 知识库检索

AI 写作时自动检索相关知识库内容并引用。MVP 阶段用简单的关键词/语义检索。架构预留向量检索切换入口。

## D7 情节设计/设定灵感记录

独立笔记空间，记录未成型的情节创意、设定灵感。MVP 用自由文本 + 标签。
