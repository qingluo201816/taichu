# 内容优化（B 类功能）

> 更新日期：2026-06-25
>
> 所属层级：**增强层（第二阶段）**
>
> 关联文档：[内容生成](./内容生成.md)、[一致性与审查](./一致性与审查.md)

## 概述

内容优化是对已生成文本的二次加工，通过聊天窗口内的操作按钮触发。B 类 Agent 在流程中调用 E 类一致性检查 Tool 来验证结果。

## B. 内容优化（聊天窗口内操作按钮）

| 编号 | 功能 | MVP 行为 | 架构预留 |
|------|------|----------|----------|
| B1 | 润色/打磨 | 直接返回结果 | LangGraph interrupt 节点（后续开启 HITL） |
| B2 | 扩写/缩写 | 直接返回结果 | 同上 |
| B3 | 风格转换 | 直接返回结果 | 同上，风格预设可配置 |
| B4 | 查错纠错 | 直接返回结果 | 同上 |
| B5 | 节奏分析 | 直接返回结果 | 同上 |

### 架构设计

- B 类每个功能是独立的 Agent/Tool
- B 类 Agent 调用 E 类一致性检查 Tool 来验证结果
- MVP 阶段跳过 HITL 节点，直接返回；后续改造只需修改节点逻辑
