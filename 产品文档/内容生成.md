# 内容生成（A 类功能）

> 更新日期：2026-06-25
>
> 所属层级：**核心层（MVP 必做）**
>
> 关联文档：[架构设计](./架构设计.md)

## 概述

内容生成是聊天窗口内的核心 AI 交互功能，每个功能作为独立 Agent 运行，有自己的 LangGraph 图和提示词。

## A. 内容生成（聊天窗口内操作）

| 编号 | 功能 | 触发方式 | 反馈方式 | MVP 行为 |
|------|------|----------|----------|----------|
| A1 | 自由对话 | 直接输入 | AI 回复 | 与 AI 对话讨论写作思路 |
| A2 | 章节续写 | 选中文字 → 点续写 | AI 返回续写内容 | 直接返回 |
| A3 | 段落生成 | 输入指令 | AI 生成内容 | 直接返回 |
| A4 | 对话生成 | 输入指令 | AI 生成对白 | 直接返回 |
| A5 | 人物描写 | 输入指令 | AI 生成描写 | 直接返回 |

### 架构设计

- 每个功能是独立的 Agent，有自己的 LangGraph 图和提示词
- API 按功能拆分，不搞一刀切
- 预留 Human-in-the-loop 节点（MVP 阶段跳过，后期打开）

## A0 会话管理（聊天窗口基础设施）

| 编号 | 功能 | MVP 范围 | 远期扩展 |
|------|------|----------|----------|
| A0-1 | 历史对话保存 | 保存对话记录到 JSON | 支持多会话切换、搜索历史 |
| A0-2 | 对话延续 | 加载历史对话继续写 | 跨会话内容关联 |
| A0-3 | 对话导出 | 远期 | 导出为 .md/.txt/.docx |
| A0-4 | Token 用量看板 | 远期 | 每次会话的 Token/费用统计可视化 |
