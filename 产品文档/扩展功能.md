# 扩展功能（远期规划）

> 更新日期：2026-06-25
>
> 所属层级：**扩展层（远期规划）**

## F. 历史与版本

| 编号 | 功能 | 说明 |
|------|------|------|
| F1 | 写作历史记录 | 查看过往写作记录 |
| F2 | 版本管理 | 同一段内容的多个版本保存对比 |
| F3 | 草稿箱 | 暂存未完成的内容 |

## G. 模板与工作流

| 编号 | 功能 | 说明 |
|------|------|------|
| G1 | 自定义提示词模板 | 保存和管理常用提示词 |
| G2 | 写作工作流 | 预设写作流程（大纲→草稿→润色→审查） |
| G3 | 快捷指令 | 常用操作一键触发 |

## H. 导入导出

| 编号 | 功能 | 说明 |
|------|------|------|
| H1 | 文件导入 | 导入 .txt/.md/.docx |
| H2 | 章节导出 | 导出单章或多章 |
| H3 | 知识库导入导出 | 批量导入导出角色/世界观数据 |

## I. 系统与配置

| 编号 | 功能 | 说明 |
|------|------|------|
| I1 | 模型切换 | 切换不同 LLM 提供商 |
| I2 | 用量统计 | Token 使用量/费用统计 |
| I3 | 写作风格预设 | 预设多种写作风格，一键切换 |
| I4 | 用户偏好设置 | 默认模型、默认风格等 |

## DX. 远期核心功能

| 编号 | 功能 | 说明 |
|------|------|------|
| DX-1 | 角色虚拟形象展示 | 主要角色以可视化形象呈现在独立页面中 |
| DX-2 | 地点 3D 动态场景 | 地点设定做成 3D 模型，可动态变换视角/状态 |

## 排除项

| 功能 | 排除原因 |
|------|----------|
| 团队协作 | 个人使用场景 |
| SEO 优化 | 与小说写作不相关 |
| 多语言翻译 | 专注于中文写作 |
| 实时协作编辑 | 个人使用场景 |
