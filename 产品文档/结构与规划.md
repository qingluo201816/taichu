# 结构与规划（C 类功能）

> 更新日期：2026-06-25
>
> 所属层级：**核心层（MVP 必做）**
>
> 关联文档：[架构设计](./架构设计.md)

## 概述

结构与规划是独立页面，提供树状大纲编辑器用于管理小说的卷/章/节结构。保存时同步知识库。

## C. 结构与规划（独立页面）

| 编号 | 功能 | 说明 | MVP 范围 |
|------|------|------|----------|
| C1 | 小说大纲 | 整体故事大纲 | 树状结构编辑器，手动编辑 |
| C2 | 章节大纲 | 单章纲要 | 树状结构子节点编辑 |
| C3 | 情节弧线 | 起承转合 | 远期内置模板，MVP 自由文本 |
| C4 | 卷/篇结构 | 分卷规划 | 树状结构顶层节点 |

### 数据模型

卷 → 章 → 节，树形嵌套结构，保存时同步知识库。
