# 产品定位与交互模型

> 更新日期：2026-06-25

## 一、产品定位

个人 AI 写作助手，专注于**玄幻小说写作**场景。以知识库为核心，提供沉浸式写作体验，以下内容都是初版，不代表最终版本，因此要有可拓展性。

## 二、用户交互模型

### 核心体验：统一聊天窗口 + 上下文操作按钮

```
  ┌──────────────────────────────────────────────────────┐
  │  小说创作助手                                        │
  │                                                      │
  │  💬 你：我想写一个主角被退婚的开头                  │
  │  🤖 AI：好创意！可以从「订婚宴现场收到退婚书」开始  │
  │  💬 你：帮我写一段这个场景(如果没说字应该弹窗给一个多少字)      │
  │  🤖 AI：（生成了一段XX字的开篇）                    │
  │                                                      │
  │  [选中文字/其他操作] ──┐                                     │
  │              │  [📝续写] [🎨润色] [⚡相关灵感生成]        │
  │              │  [📏扩写] [📏缩写] [🔍查错]         │
  │              │  [🔎搜设定] [📊检查设定]             │
  │              └─────────────────────────────────────  │
  └──────────────────────────────────────────────────────┘
```

### 辅助页面：独立管理视图

| 页面 | 用途 | 交互 |
|------|------|------|
| **大纲/目录** | 树状结构管理卷/章/节，宏观把控 | 独立页面，未来要找一个设计感的方案 |
| **知识库-角色** | 角色卡片管理 | 独立页面，表单+列表 |
| **知识库-世界观** | 世界设定笔记 | 独立页面，分类+自由文本 |
| **知识库-功法** | 功法百科 | 独立页面，列表+详情 |
| **知识库-地点** | 地点/场景记录 | 独立页面，列表+详情 |
| **知识库-时间线** | 事件时间轴 | 独立页面，时间线视图 |
